# Babylon.js Solar System Explorer

Ce projet est une expérience interactive 3D du système solaire réalisée avec Babylon.js et Vite. Il permet d’explorer les planètes dans une scène 3D, de naviguer entre elles et d’afficher des informations détaillées via une interface simple.

## ✨ Fonctionnalités

- Visualisation interactive du système solaire en 3D
- Navigation entre les planètes avec boutons et interaction sur la scène
- Affichage du nom de la planète actuellement sélectionnée
- Popup d’informations détaillées sur chaque planète
- Interface légère et responsive

## 🛠️ Technologies utilisées

- [Babylon.js](https://www.babylonjs.com/) pour la scène 3D
- [Vite](https://vitejs.dev/) pour le développement local
- HTML, CSS et JavaScript vanilla

## ▶️ Installation

1. Clonez le dépôt :

   ```bash
   git clone https://github.com/your-username/babylon-project.git
   cd babylon-project
   ```

2. Installez les dépendances :

   ```bash
   npm install
   ```

3. Lancez le projet en mode développement :

   ```bash
   npm run dev
   ```

4. Ouvrez l’URL affichée par Vite dans votre navigateur.

## 🎮 Utilisation

- Cliquez sur une planète pour la sélectionner
- Utilisez les boutons précédent/suivant pour naviguer
- Cliquez deux fois rapidement sur une planète pour ouvrir sa fiche d’information

## 📌 Notes

Le projet utilise un modèle 3D du système solaire et des textures associées, disponibles dans les dossiers du projet.
